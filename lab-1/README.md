# Лабораторная №1

`Вариант 368849`

1. Для функции `sec(x)` провести модульное тестирование разложения функции в степенной ряд. Выбрать достаточное тестовое покрытие.
2. Провести модульное тестирование сортировки пузырьком. Для этого выбрать характерные точки внутри алгоритма, и для предложенных самостоятельно наборов исходных данных записать последовательность попадания в характерные точки. Сравнить последовательность попадания с эталонной.
3. Сформировать доменную модель для заданного текста.  Разработать тестовое покрытие для данной доменной модели:

```text
Артур неожиданно почувствовал, как на его бесплотной голове зашевелились волосы, когда он начал медленно, но неуклонно двигаться к пульту, но это был всего лишь, как он тут же догадался, наезд камеры при съемке.
```
